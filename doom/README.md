# Doom Raycasting Engine

A 3D raycasting engine using DDA (Digital Differential Analyzer) for efficient ray-grid traversal.

## Compile & Run

```bash
cd doom && gcc doom.c kengine.c -o doom -lSDL3 -lm && ./doom
```

## Controls

- **W/UP**: Forward  
- **S/DOWN**: Backward  
- **Q/LEFT**: Rotate left  
- **E/RIGHT**: Rotate right
